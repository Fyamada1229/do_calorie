## :orange_book: カロリー計算アプリ
誰でも簡単にカロリー記憶をつけることができるアプリです。<br>
ストレスなく誰でも使えるをコンセプトにしてUIを意識して作りました。<br>
ユーザー登録、ログイン機能、カロリー計算、などの機能があります。<br>

## :orange_book: 実装した機能
- ユーザー登録
- ログイン機能
- カロリー計算機能
- 編集機能
- マイページ機能
- カレンダー機能
